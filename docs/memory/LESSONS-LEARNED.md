## 💎 1. กลยุทธ์การเทรด (Trading Strategy)
### ✅ การกู้ชีพพอร์ตด้วย "ความงก" (Greedy Scalper)
- **Lesson:** การตั้งเป้ากำไรสูงเกินไปในตลาดที่มีความผันผวน (Fear Zone) ทำให้กำไรทิพย์หายบ่อย (กรณี MOVR +5% ร่วงกลับมาเหลือ 0%)
- **Solution:** ปรับใช้ **Dynamic Trailing Lock** (ถอย 0.2% จาก Peak เมื่อกำไรพ้นค่าธรรมเนียม) ช่วยสะสมกำไรไม้เล็กๆ ได้จริง
- **Success Case:** ORCA-USDT เก็บกำไรสะสมได้หลายรอบ และรันเทรนด์ได้ถึง +6.61% เมื่อระบบ Dynamic เปิดช่องว่างให้หายใจ

### ⚠️ วินัยการตัดขาดทุน (Stop Loss Management)
- **Lesson:** Stop Loss ที่กว้างเกินไป (-7%) ไม่เหมาะกับพอร์ตขนาดเล็กในโหมด Scalping เพราะกำไรที่สะสมมาอาจหายหมดในการแพ้ครั้งเดียว
- **Solution:** ใช้ **Hard Cut-Loss -1.5%** ร่วมกับระบบ **Break Even** (ขยับ SL มาที่ทุนเมื่อกำไรถึง 1%) เพื่อปกป้องเงินต้น

## 🛠️ 2. การจัดการระบบ (System & DevOps)
### 🔐 ความลับและการรันบอท (Infisical + PM2)
- **Lesson:** การ Restart PM2 แบบปกติไม่สามารถโหลด Environment Variables ใหม่จาก Infisical ได้ทันที
- **Solution:** ต้องรันผ่าน `infisical run -- pm2 start ecosystem.config.js` เพื่อให้มั่นใจว่ากุญแจ API จะถูกฉีดเข้าสู่ Process ลูกอย่างถาวร

### 🧹 การจัดการขยะ (Python Cache Issues)
- **Lesson:** บางครั้งการแก้ไขโค้ดไม่ส่งผลเพราะ Python รันไฟล์ `__pycache__` ตัวเก่า
- **Solution:** ต้องทำ **Deep Clean** ลบโฟลเดอร์ `__pycache__` ทุกครั้งที่มีการแก้ไข Logic สำคัญในสมอง
